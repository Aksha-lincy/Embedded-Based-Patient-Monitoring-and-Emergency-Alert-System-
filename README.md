# Embedded-Based-Patient-Monitoring-and-Emergency-Alert-System-
Embedded-Based Patient Monitoring and Emergency Alert System is an ATmega328-based healthcare device that monitors heart rate and sweat levels. The monitored data is displayed on a 16×2 LCD. When abnormal conditions are detected, a buzzer alert is activated, ensuring timely attention and improved patient safety.
The Embedded-Based Patient Monitoring and Emergency Alert System is a healthcare monitoring device developed to continuously observe a patient’s health condition and provide immediate alerts whenever abnormal conditions are detected. The system is designed using the ATmega328 microcontroller, which acts as the central processing unit for monitoring and controlling all system operations. The main objective of the project is to improve patient safety by enabling real-time monitoring and rapid emergency notification.

The power supply section consists of an AC mains source, a step-down transformer, a bridge rectifier, a capacitor filter, and a voltage regulator. This arrangement converts the AC supply into a stable DC voltage required for the proper operation of the microcontroller and other electronic components. A 16×2 LCD display is interfaced with the microcontroller to display monitoring information and system status in real time.

The system monitors two important health-related parameters: heart rate and sweat level. For demonstration purposes, a potentiometer is used instead of an actual heart rate sensor to simulate different heart rate conditions. A sweat sensor is incorporated to detect excessive perspiration, which may indicate stress, fever, anxiety, dehydration, or other abnormal physiological conditions. The ATmega328 continuously receives and processes the sensor data and compares the values with predefined threshold levels.

Whenever the simulated heart rate exceeds the safe limit or excessive sweating is detected, the microcontroller activates a buzzer through an NPN transistor driver circuit. The buzzer provides an audible alert to notify nearby caregivers, family members, or medical personnel, allowing them to respond quickly to the patient’s condition. This immediate alert mechanism helps reduce response time and improves patient safety during emergency situations.

The proposed system is simple, reliable, cost-effective, and easy to implement. It can be used in hospitals, clinics, elderly care centers, and home healthcare environments for basic patient monitoring applications. The project demonstrates the practical application of embedded systems in healthcare technology and provides a foundation for future enhancements such as wireless communication, IoT integration, mobile notifications, and remote health monitoring. Overall, the system offers an efficient solution for continuous patient monitoring and emergency alert generation.
